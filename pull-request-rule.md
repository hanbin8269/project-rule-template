# Pull Request 규칙

## 제목
- `#<이슈번호> 수행한 일` 형식으로 적습니다.
`ex) #12 I killed KSH`

- 이슈를 해결하는 PR이 아니라면 수행한 일만 간결하게 적습니다.
`ex) I killed KSH`


## 본문
- 만약 이슈를 처리한 PR이라면 이슈 링크를 위해서 본문에 `Fixed #<이슈번호>`를 꼭 넣어줍시다. 
`ex) Fixed #12`
